# Post-Call Webhook

The post-call webhook delivers real-time notifications and detailed call data after each call ends, enabling powerful integrations and analytics.

<Card title="Key Benefits" icon="star">
  - Real-time call data delivery
  - Rich insights for business intelligence
  - Integration with existing systems
  - Enables automated workflows and actions
</Card>

## Webhook Data

<AccordionGroup>
  <Accordion title="Call Metadata">
    - Call duration
    - Caller and recipient information
    - Call status (completed, dropped, etc.)
  </Accordion>
  <Accordion title="Conversation Analysis">
    - Sentiment analysis
    - Key topics discussed
    - Action items identified
  </Accordion>
  <Accordion title="Agent Performance">
    - Response times
    - Adherence to script
    - Customer satisfaction indicators
  </Accordion>
</AccordionGroup>

## Security Considerations

<Warning>
  To ensure the security of your webhook:
  - Use HTTPS URLs for your endpoint
  - Implement proper authentication
  - Regularly rotate your webhook secrets
</Warning>

## Configuring the Webhook

<Steps>
  <Step title="Navigate to Settings">
    Go to "Agent" > "Setting" > "General" in your control panel
  </Step>
  <Step title="Find Webhook Section">
    Locate the "Webhook" section
  </Step>
  <Step title="Enter Webhook URL">
    Input your secure HTTPS webhook URL
  </Step>
  <Step title="Save Configuration">
    Click "Save" to activate your webhook
  </Step>
</Steps>

For detailed API specifications and data schemas, refer to our [API Documentation](/api-reference/introduction).
