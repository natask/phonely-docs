# Send SMS Block

**Purpose**: Send text messages during or after calls for follow-up and notifications.

**What It Does:**
- Composes and sends SMS messages
- Interpolates call variables into message content
- Tracks delivery status and confirmation
- Supports scheduling for future delivery

**Key Features:**
- **Message Content**: Dynamic text using collected variables
- **Recipient Selection**: Send to caller or other numbers
- **Scheduling**: Send immediately or schedule for later
- **Template Support**: Use predefined message templates
- **Delivery Confirmation**: Track message delivery status

**Configuration Options:**
- **Message Text**: Content of the SMS
- **Recipient**: Phone number to send to
- **Scheduling**: When to send the message
- **Template**: Predefined message template
- **Delivery Tracking**: Whether to track delivery status

**Testing Tips:**
- Test message sending with various content
- Verify variable interpolation in messages
- Check delivery confirmation and status
- Validate scheduling and timing

**Common Use Cases:**
- Appointment confirmations
- Order status updates
- Follow-up reminders
- Emergency notifications
- Marketing messages
